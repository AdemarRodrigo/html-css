Tags de Estrutura Básica do Site

Header

O elemento representa um contêiner para conteúdo introdutório ou um conjunto de links de navegação.

Nav

A tag define um conjunto de links de navegação

Section

A tag define uma seção em um documento.

Article

tag especifica conteúdo independente e autocontido.

Aside

A tag define algum conteúdo além do conteúdo em que é colocado.

Footer

A tag define um rodapé para um documento ou seção.
